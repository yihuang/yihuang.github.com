显然正确的代码

应该贴近自然语言描述

问题1

从列表取大于10且小于100的数

表达： 从列表取 ? 的数

λ x -> filter ? x

不如直接点：

filter ?

表达： 大于10

λ x -> x > 10

不如直接点：

(>10)

表达： 小于100

同样：

(<100)

表达： 且

&&

:: Bool -> Bool

类型不对

表达： 且

?

   (a -> Bool)
-> (a -> Bool)
-> (a -> Bool)

表达： 且

liftA2 (&&)

   (a -> Bool)
-> (a -> Bool)
-> (a -> Bool)

表达： 且

(&&&) = liftA2 (&&)

拼在一块：

filter ( (>10) &&& (<100) )

>>> let foo = filter ( (>10) &&& (<100) )
>>> foo [1..20]
[11, 12, 13 ... ]
